Estructura de directorios es la siguiente:
admin -> directorio administración
	config.php -> archivo de configuración
	Subdirectorios->contienen la información de los diferentes gestores	
frontend-> directorio de frontend
	CSS->style-css
	imagenes->img
	cuenta de usuario->userAccount
Management->management
lib->directorio de librerias compartidas
sim->directorio donde está ubicado el simulador
index.php-> página principal
news.php->noticias
events.php->eventos
